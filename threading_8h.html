Detailed Description

Threading helpers.

Contains low-level TLS helper routines and wrappers for creating new Besta RTOS threads that work well with EABI and TLS.

int mutekix_thread_wrapper ( void *  arg)
extern

Wrap the thread function for ABI and THUMB compatibility.

This wrapper ensures the stack alignment and uses BX to invoke the real function to properly set the THUMB bit when jumping to it.

Parameters
argThe real function and its user data pointer.
Returns
Thread exit code.

◆ mutekix_tls_init()

int mutekix_tls_init ( thread_t *  thr)
extern

Initialize Thread Local Storage (TLS) for a thread.

This allocates the TLS block (if applicable) and set all values in that block to NULL.

This must be called after the thread is created with OSCreateThread() if TLS will be used later.

Parameters
thrThread descriptor the TLS block belongs to.
Returns
0 if successful.
