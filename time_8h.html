Detailed Description

Utilities for time measurement.

There is no standard way to measure time elapsed at millisecond-level on Besta RTOS. There is a millis field in the datetime_t struct but its behavior is not standard across devices. The purpose of this component is to provide a standard way of time measurement by choosing the best measurement method for each device, handling the device-specific behavior required by each known device type, and trying to fall back to a safe default if unexpected behavior is detected.
