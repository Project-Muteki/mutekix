26/* Coverage helper header.
27 Computing coverage for a generic macro expansing library is not trivial
28 as there is no tool to do it normally.
29
30 That's why a special way has been designed:
31 *) Put the code to coverage within the macros START_COVERAGE & END_COVERAGE
32 (without forgetting to include the header coverage.h before)
33 #include "coverage.h"
34 START_COVERAGE
35 [code to be expanded & to be coveraged]
36 END_COVERAGE
37
38 Run the coverage in the makefile like this:
39 *) make coverage-test/test-marray.c
40
41 View the coverage file test-marray.c.gcov
42 It'll contain the macro expanded & annotated.
43*/
